# YourBlog App

A project that mimics the features of a blog web-app.

## Built With
- HTML/CSS
- Ruby
- Ruby on Rails
- Postgresql

## Getting Started
To get this program running on your computer, <br>
1.Install Ruby<br>
2.Clone the project repo 

### Run (_**terminal**_)
For **dependencies**, run
```
  $ bundle install
  $ npm i
```
For **tests**, run
```RSpec
  $ rspec test
```

For **Rubocop** fixes, run
```rubocop
  $ rubocop -A
```
